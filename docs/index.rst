Pinecone's Python SDK Reference
==========================================================

`Pinecone <https://www.pinecone.io/>`__ provides similarity search as a managed service.

Learn more about Pinecone and how your ML models can benefit from a similarity search 
service at `pinecone.io/docs <https://www.pinecone.io/docs/>`__ and 
`pinecone.io/learn <https://www.pinecone.io/learn/>`__.

üìù Questions and suggestions: `pinecone.io/contact <https://pinecone.io/contact/>`__


Installation
------------------

.. code-block:: bash
    
    pip install -U pinecone-client


.. tip::
   Pinecone is tested on Python 3.6+.

   We strongly recommend that you install Pinecone in a virtual environment.
   We think it's good for Python hygiene. 
   It's really easy to start using virtual environments with one of these guides: 
   `[1] <https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/>`__,
   `[2] <https://realpython.com/python-virtual-environments-a-primer/>`__.



Getting Started
------------------

To get started, visit `pinecone.io/start <https://www.pinecone.io/start/>`__ to get an API key.

Once you have received your API key in the email,
head over to `pinecone.io/docs <https://www.pinecone.io/docs/>`__ to see examples and tutorials.
